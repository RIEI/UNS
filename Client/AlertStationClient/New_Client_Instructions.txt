1.) Move the AlertStationClient folder to c:\
2.) Modify "StartClient.bat". Set the homepage to your uns url (eg. http://unsserver/uns/index.php?id=4c6a8a77381e6e6f41a536ef50891279 )
3.) Modify the url stored in the UNS chrome plugin. edit "AlertStationClient\Chromium\_UNS-Extension\background.js". set alertxml to the uns xml url (eg. http://unsserver/uns/index.php?id=4c6a8a77381e6e6f41a536ef50891279&out=xml )
4.) Log into the user you want to run UNS as and set uns to automatically start by running "AlertStationClient\uns_client_shell.reg"
5.) Set default cursors to blank or a small dot(AlertStationClient\tinycursors) so they don't show on the sceen.
6.) Use Autologon "AlertStationClient\Autologon" to make the computer automatically log on as the user you set up.
